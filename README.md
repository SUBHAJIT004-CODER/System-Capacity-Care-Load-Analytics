
# Healthcare Capacity & Care Load Analytics

## Project Overview

This project analyzes healthcare system capacity and care load trends for unaccompanied children using historical custody and care data.

## Features

- Data Analysis
- KPI Dashboard
- Trend Analysis
- Correlation Analysis
- AI Forecasting using Random Forest
- Streamlit Web Application

## Dataset Columns

- Date
- Children apprehended and placed in CBP custody
- Children in CBP custody
- Children transferred out of CBP custody
- Children in HHS Care
- Children discharged from HHS Care

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- Streamlit

## Model

Random Forest Regressor

## Deployment

Deploy using Streamlit Community Cloud connected to GitHub.
